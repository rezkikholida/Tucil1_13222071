# IF2211_Tucil1_13222071
## Deskripsi Program - Penyelesaian Permainan Queens Linkedin
Sebuah program yang bertujuan untuk menyelesaikan permainan Queens Linkedin. Program ini menerima masukan posisi warna, lalu memberikan keluaran posisi queens sedemikian rupa sehingga memenuhi syarat yaitu terdapat hanya satu queen pada tiap baris, kolom, dan daerah warna. Selain itu, satu queen tidak dapat ditempatkan bersebelahan dengan queen lainnya, termasuk secara diagonal.

## Requirement Program
- Python
- Tkinter
- numpy

## Cara Compile Program
1. Untuk Python versi windows, umumnya pip dan Tkinter sudah terpasang bersama dengan instalasi Python.
2. Buka repository dengan terminal, kemudian masuk ke directory `source_code`
3. Compile program utama

## Cara Menjalankan Program
1. Program dapat dijalankan mengunduh repository lalu dibuka menggunakan compiler seperti VScode
2. Jalankan kode dengan mengklik tombol run
3. Hasil diperoleh pada GUI dan terminal
